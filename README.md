# Проект: Место

Место - это одностраничный сайт, интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки. 

### Использованные технологии:

- Флексбокс
- БЭМ Nested
- Адаптивная вёрстка для 3 различных разрешений экрана
- Гриды
- JS
